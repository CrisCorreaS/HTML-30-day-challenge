Atajos de Windows que utilizo a diario

  • Windows + E → Abre el Explorador de Archivos
  • Ctrl + Shift + N → Crea una nueva carpeta
  • Windows + L → Bloquea el ordenador o cambiar de cuenta
  • Windows + G → Abre el Game DVR para grabar
  • Ctrl + F → Permite buscar o abrir un buscador
  • Windows + M → Minimiza todas las ventanas
  • Windows + Shift + S → Abre la herramienta de recortes
  • Windows + D → Muestra y oculta el escritorio
  • Windows + Ctrl + D → Crea un nuevo escritorio virtual
  • Alt + Tab → Permite cambiar entre las aplicaciones abiertas.
  • Windows + Tab → Accede a todos los escritorios y aplicaciones con el Visor de Tareas
  • Windows + Derecha o Windows + Izquierda → Ancla la ventana activa a la derecha o a la izquierda
